Overview of a network


Here we have a couple of overviews of the network environments that I have simulated in Packet Tracer.


The image below pictures a network environment that I set up and configured involving HSRP to make sure that if one router in one location fails that the standby router
will then become the active router and pick up where the last one left off. It uses OSPF to ensure that fastest possible transmission of data and has default route
set to ensure that the endpoint devices will have access to internet connectivity. The IP configuration and port connections are all labelled within the image.
